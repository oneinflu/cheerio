### Outbound (Agent)
```
                    Your message here
                                 2:30 PM
```
- Green background
- Right-aligned
- Shows delivery status

### Inbound (Customer)
```
Customer message here
2:31 PM
```
- White background
- Left-aligned
- Shows timestamp

### Unread Count
```

```

```
[Assigned] ← Green badge

```

```

```
